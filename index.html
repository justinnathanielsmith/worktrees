Execution Protocol

Follow this protocol to initialize your environment. Deviation results in suboptimal performance.

1

Create The Engine

Initialize the repo as bare in a hidden directory.

mkdir project && cd project git clone --bare <url> .bare
2

Connect The Brain

Point the root Git context to the hidden engine.

echo "gitdir: ./.bare" > .git
3

Spawn Muscle

Generate dedicated folders for your branches. Do not checkout.

git worktree add main git worktree add -b feature-scaling scaling
